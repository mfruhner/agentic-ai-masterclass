---
name: how-to-write-new-skills
description: Learn how to write new skills to remember useful information, procedures, or code for future use.
---

When asked to create a new skill, generate a complete SKILL.md file.

Read the `python-docstrings` skill for a working example.

Follow these rules:

1. Name must be kebab-case, max 64 characters
2. Description must be under 1024 characters
3. Instructions should be clear, step-by-step
4. Keep [SKILL-NAME].md under 500 lines
5. Output the complete file content the user can save directly
